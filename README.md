# ShenEternity-Agent

一个基于 Rust 构建的终端 LLM Agent客户端，使用 `ratatui` 和 `crossterm` 打造现代化的 TUI 界面。
目前只写完聊天功能。

## ✨ 特性

- 🖥️ **终端 UI 界面** - 基于 ratatui 构建的美观终端界面
- 💬 **流式聊天** - 支持流式接收 LLM 响应，实时显示
- ⌨️ **键盘驱动** - 完全基于键盘操作，高效便捷
- 🎨 **彩色界面** - 丰富的颜色主题和边框样式
